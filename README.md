# Starforce Simulator

A fully client-side web simulator for MapleStory's Starforce enhancement system.  
This tool runs thousands of simulated enhancement attempts and visualizes the results with charts, percentiles, ranges, and statistical summaries.

 **[Try it out!](https://wizhe.github.io/sf_simulator/)** (https://wizhe.github.io/sf_simulator/)

## Features

- Simulate thousands of Starforce enhancement runs entirely in the browser
- **Configurable parameters**:
  - Item level  
  - Starting and target stars  
  - Number of simulation runs  
  - Starcatch  
  - Safeguard  
  - 30% off event  
  - 30% boom chance reduction event  

- **Cost statistics**:
  - Average & median cost  
  - Cost range  
  - Percentiles
  - Cost Histogram

- **Boom statistics**:
  - Average & median booms  
  - Boom range  
  - Percentiles  
  - Boom histogram
